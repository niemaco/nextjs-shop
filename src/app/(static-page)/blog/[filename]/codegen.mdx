# GraphQl codegen

Codegen, GraphQL Codegen lub GraphQL Code Generator from The Guild company

[Website](https://the-guild.dev/graphql/codegen)

Tools to generate types basic on schema and GraphQl request

## Configuration

Install cli wizard
```
pnpm add -D @graphql-codegen/cli
```

next init generator

```
pnpm graphql-code-generator init
```
